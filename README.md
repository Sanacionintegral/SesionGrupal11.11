# Sitio estático — Sesión grupal (v2)

Incluye páginas:
- `index.html`: bienvenida, precio promocional y leyenda especial por **Portal 11·11**.
- `pagar.html`: datos para transferir (**CVU, Alias, Titular**), copiar y WhatsApp a Clara.
- `acceso.html`: acceso posterior al pago con links a **Ficha** e **Instrucciones** (páginas HTML).
- `instrucciones.html`: el texto completo de instrucciones (no descarga PDF).
- `ficha.html`: botón que abre tu **Formulario de Google** (reemplazar URL).
- `styles.css`: estilos.

## Pasos
1. Subir al repo y activar **GitHub Pages** (branch main / root).
2. En `ficha.html`, reemplazar `https://forms.gle/XXXXXXXXXXXXXXX` por tu enlace real.
3. (Opcional) Cambiar número de WhatsApp en `pagar.html`.
4. Para habilitar acceso sin depender del botón, usar `acceso.html?clave=OK`.
